# Customer Sales & Revenue Analysis Dashboard

## Project Overview

This project analyzes a dataset containing **3,900 customer records** to uncover business insights related to sales performance, customer behavior, and revenue trends. The analysis combines **Python**, **SQL**, and **Power BI** to perform data exploration, answer business questions, and create an interactive dashboard for decision-making.

---

## Objectives

* Perform exploratory data analysis (EDA) to understand customer purchasing patterns.
* Clean and preprocess the dataset for accurate analysis.
* Use SQL to answer key business questions and generate actionable insights.
* Develop an interactive Power BI dashboard to visualize sales and revenue performance.

---

## Dataset

* **Rows:** 3,900 customer records
* **Contents:** Customer information, product purchases, sales transactions, revenue, and related business metrics.

---

## Tools & Technologies

| Tool         | Purpose                                                           |
| ------------ | ----------------------------------------------------------------- |
| **Python**   | Data cleaning, preprocessing, and exploratory data analysis (EDA) |
| **SQL**      | Querying the dataset and answering business questions             |
| **Power BI** | Building interactive dashboards and visualizations                |

---

## Project Workflow

### 1. Data Exploration (Python)

Using Python libraries such as **Pandas** and **NumPy**, the dataset was:

* Cleaned and preprocessed
* Checked for missing values and duplicates
* Explored using descriptive statistics
* Feature Engineering
* Analyzed to identify trends, distributions, and correlations

---

### 2. Business Analysis (SQL)

SQL was used to answer business-oriented questions, including:

* Total revenue generated
* Categorised customers based on purchase frequency
* Revenue by product category
* Top performing products
* Customer purchasing behavior
---

### 3. Dashboard Development (Power BI)

An interactive Power BI dashboard was created featuring:

* Revenue and Sales Overview
* Subscription Status of consumers
* Customer Analysis
* Category-wise Revenue
* Interactive filters and slicers for dynamic reporting

---

## Key Insights

* Identified the highest revenue-generating products.
* Categorised customers into three broad categories based on purchase frequency.
* Analyzed demographic patterns.
* Evaluated category-wise revenue performance.
* Enabled interactive exploration of business metrics through Power BI.
